1. Explain how CSS differs from HTML in the context of web development.
A. CSS differs from HTML because they both served specific functions. CSS controls styling and layout of the web page, while HTML builds the structure. CSS uses the elements defined in HTML to style each component.  

2. List the methods of including CSS in an HTML document and briefly describe each.
A. There are 3 methods to include CSS in an HTML doc.
First is through inline styling, adds a style directly to a selector within the HTML file. 
Second is by done by adding an opening and closing style tag on the HTML file and writing CSS directly in the HTML file within the style tags.
Third is using an external sheet and connecting it using the link tag. This is most popular. 

3. Describe the anatomy of a CSS rule.
A. Selector, declarations, properties and values.

4. When might you want to use RGBA instead of RGB?
A. When you want to add transparency/opacity to your styling. 

5. How do Hex color values differ from RGB?
A. They differ in the way they are written/formatted. Hex uses a combination of 6 letters and numbers. RGB uses 3 values to represent hue of red, green and blue. 

6. When might a developer prefer to use HSL over other color formats?
A. HSL is the preferred method to be used when adding transparency vs. RGBa like shadows or gradiants.

9. Explain the difference between em and rem units.
A. REM inherits from the root element. EM inherits from it's nearest parent element.
